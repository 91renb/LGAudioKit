##添加到工程
- 拷贝'Class'文件夹到你的工程
- #import "LGAudioKit.h"

##说明
- Demo中包含两个控件：

    录音控件：'LGSoundRecorder'

    播放控件：'LGSoundPlayer'
- 具体的使用细节请参考Demo中的ViewController.m

##环境支持
- iOS7及以上
